# Metristics-SOEN6611
Metristics project for SOEN6611 - Software Measurement course, Summer 2023, Concordia University

This project aims to use the Java programming language to implement METRICSTICS from scratch.

### Salient features
1. Upload CSV file with numbers in range [1-1000]
2. Calculate Minimum of these elements
3. Calculate Maximum of these elements
4. Calculate Mean of these elements
5. Calculate Mode of these elements
6. Calculate Median of these elements
7. Calculate Mean Absolute Deviation of these elements
8. Calculate Standard Deviation of these elements

### Features Used
1. Java Swing for GUI
2. Events for handling UI actions
3. ArrayList for parsing and storing input data
4. IOException to handle errors
